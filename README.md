# Suds On Wheels Website

Static two-page website for a mobile dog bath truck business.

## Pages
- `index.html` (home)
- `services.html` (packages, service area, FAQ)

## Local preview
Open `index.html` directly in a browser, or run any static file server.

## Deploy to GitHub Pages
1. Create a new GitHub repository and push these files.
2. In GitHub: `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Choose branch `main` and folder `/ (root)`.
5. Save. Your site will publish at your GitHub Pages URL.
